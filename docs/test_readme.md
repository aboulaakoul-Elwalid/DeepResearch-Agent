# Test Document

This is a test document for the local docs RAG feature.

## Overview

The DR-Tulu agent can now search through your local documents using semantic search.
Simply drop your .txt, .md, or .pdf files into the docs folder and run ingest_local.py.

## Features

- Supports .txt, .md, and .pdf files
- Uses sentence-transformers for embeddings
- Stores in ChromaDB for fast retrieval
- Integrates with DR-Tulu MCP tools
